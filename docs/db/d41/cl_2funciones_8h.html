Pantalla de carga de IP.

Parámetros
cvarEstructura con las variables de allegro.
cvarclEstructura con las variables de red del cliente.
Devuelve
3 para que mantenga la pantalla de carga de ip, 1 para ir al menu, 0 para jugar.

En esta pantalla el jugador puede escribir la IP del servidor al que se quiere conectar. Debe escribir la IP con puntos y apretar ENTER. Permite borrar en caso de cargar mal la direccion. Tambien puede volver al menu.
